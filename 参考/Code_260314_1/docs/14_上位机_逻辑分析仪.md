## 控制命令

| 命令 | UI 控件 | FPGA 行为 |
|:--:|------|---------|
| 0x60 | 采样率下拉框 | 设置 `la_sample_div` (50MHz/div) |
| 0x62 | 触发配置面板 | 设置触发使能 + 掩码 + 值 |
| 0x63 | 启动按钮 | `la_capture_en` 单周期脉冲 |
| 0x64 | 停止按钮 | `la_capture_stop` 脉冲 |

> 0x61 (采集长度) 上位机未使用，FPGA 强制 `la_capture_len = 0` (连续采集)。

## DSA 面板 — 数字信号分析

**源文件**: `src/APP/logic_analyzer/digital_signal_panel.py`

```
功能: 对 DIGITAL_IN[7:0] 8 通道进行频率和占空比自动测量

操作:
  0x66: 开始 8 路测量
  0x67: 停止测量
  0x68: 读取结果 (单选通道)

接收: dsa_measurement_received Signal
  → 更新界面: 频率 / 高电平周期 / 低电平周期 / 占空比
```

### DSA 数据格式

```
0x68 返回: 20 字节 = 7 字节应答帧 + 13 字节数据

应答帧: AA 55 01 68 00 00 CS
数据帧: [channel 1B][freq LE32 4B][high_cycles LE32 4B][low_cycles LE32 4B]

上位机解析:
  freq_hz = 50_000_000 / (high_cycles + low_cycles)  # 近似
  duty_pct = high_cycles / (high_cycles + low_cycles) * 100
```

## PulseView .sr 导出

**源文件**: `src/APP/utils/pulseview_exporter.py`

将采集到的逻辑分析仪数据导出为 sigrok PulseView 兼容的 `.sr` 文件格式：

```python
# .sr 文件结构:
# - metadata (JSON): 通道数, 采样率
# - logic-1-1 (二进制): 原始采样数据 (LSB-first)
```

导出的文件可在 PulseView 软件中打开，进行协议解码 (I2C/SPI/UART 等)。
