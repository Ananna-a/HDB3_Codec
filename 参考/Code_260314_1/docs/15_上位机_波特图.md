## 上位机信号订阅

```python
class BodePlotterTab(QWidget):
    def __init__(self, serial_manager):
        self.serial_manager = serial_manager
        # 订阅 Bode 数据流
        self.serial_manager.bode_data_received.connect(
            self._on_bode_data
        )
    
    def _on_bode_data(self, freq_index, freq, mag_ch1, phase_ch1, 
                       mag_ch2, phase_ch2):
        """接收每个频点的 IQ 解调结果"""
        self.magnitude_data.append((freq, mag_ch1))
        self.phase_data.append((freq, phase_ch1))
        self._update_plots()
```

## 设计预期 (FPGA 实现后)

当 `Bode_Analyzer` 模块实现后:

```
上位机 0xB0 配置扫频 → 0xB1 启动
    ↓
FPGA Bode_Analyzer:
  1. DDS 通道A 接管: 输出正弦扫频激励
  2. ADC 双通道同步采集 (CH1=激励, CH2=响应)
  3. IQ 解调 (CIC滤波器): 提取幅度比 + 相位差
  4. 结果通过 CH340 → uart_tx_mux (Bode 通道, 高优先级) → 上位机
    ↓
上位机 BodePlotterTab:
  逐点接收 + 更新幅频/相频图
  扫频完成 → 显示完整曲线
```

### 跨时钟域数据同步

ADC 50MHz 域数据需同步到 125MHz DDS 域的 Bode 模块:

```verilog
// src/debugger_top.v:296-351
// 2 级同步寄存器 (亚稳态防护)
adc_ch1_data_sync1 <= adc_ch1_data;  // 50MHz 域捕获
adc_ch1_data_sync2 <= adc_ch1_data_sync1; // 125MHz 域稳定输出
```

> ⚠️ 当前数据同步方式可能采样到 50MHz 变化的中间值，但对 Bode 分析（统计平均 I/Q 解调）影响较小。
