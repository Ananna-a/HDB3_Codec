# 上位机 — 协议转换器 Tab

**源文件**: 8 个面板文件，组合在一个容器 Tab 内

---

## 架构

协议转换器使用嵌套 Tab 结构：

```
ProtocolConverterTab (容器, QTabWidget)
  ├── Tab 0: 序列发生器 (logic_analyzer_tab.py)
  ├── Tab 1: PWM 控制器 (pwm_controller_tab.py)
  └── Tab 2: 设备中心 (device_center_tab.py, 嵌套 QTabWidget)
        ├── CAN 面板 (can_panel.py)
        ├── I2C 面板 (i2c_panel_simple.py)
        ├── SPI 面板 (spi_panel_simple.py)
        └── DS18B20 面板 (ds18b20_panel.py)
```

所有面板接收 `SerialManager` 实例，通过 `send_command()` 与 FPGA 通信，通过订阅信号接收数据。

## 序列发生器 Tab

**源文件**: `src/APP/protocol/logic_analyzer_tab.py`

### 两种操作模式

| 模式 | 命令区间 | 界面 |
|------|:--:|------|
| 并行模式 | 0x30-0x34 | 频率滑块 + 数据编辑器 (N 字节) + 启动按钮 |
| 串行模式 | 0x40-0x43 | 通道选择 (0-7) + 独立频率 + 逐字节序列编辑 |

### 新协议 (0x40-0x43) 操作流程

```
1. 选择通道 (0-7)
2. 0x40: 配置频率+长度 (独立参数)
3. 0x41: 逐字节写入序列数据 (地址 0-255)
4. 0x42: 设置通道掩码 (使能/禁用)
5. 0x43: 可选全局复位

实时播放: 配置后自动开始按频率步进
```

## PWM 控制器 Tab

**源文件**: `src/APP/protocol/pwm_controller_tab.py`

### 8 通道独立控制

```
每个通道独立配置:
  频率: 1 Hz – 1 MHz (可输入数值或滑块)
  占空比: 0% – 100% (滑块)
  使能: 复选框

全局控制:
  启动/停止按钮 (0x51/0x52)
```

### 发送协议

```python
# 配置通道 0: 1kHz, 50% 占空比
payload = struct.pack('>BIB', 0, 1000, 32768)  # 通道ID + 频率 BE32 + 占空比 BE16
serial_manager.send_command(0x50, payload)

# 使能通道 0
serial_manager.send_command(0x51, bytes([0x01]))  # 掩码 bit0=CH0
```

## CAN 面板

**源文件**: `src/APP/protocol/can_panel.py`

### 功能

```
发送:
  选择帧类型 (标准/扩展)
  输入 ID (11/29 位)
  输入数据 (0-8 字节十六进制)
  → send_command(0xC1, payload)

接收:
  订阅 can_data_received Signal
  显示接收到的帧: ID + DLC + 数据 + 时间戳
  
配置:
  0xC0: 波特率选择 (1M/500k/100k/10k/5k)
  0xC2: 过滤器设置 (预留)
```

### 发送帧格式

```python
# 标准帧: ID=0x456, DLC=3, data=[0xAA, 0xBB, 0xCC]
payload = bytes([0x00])        # type=标准帧
payload += bytes([0x04, 0x56]) # ID_H, ID_L|DLC → (0x04<<5)|(0x56>>3)=0x8A, (0x56<<5)|3=0xAC3
payload += bytes([0xAA, 0xBB, 0xCC])
```

### 接收数据识别

```python
# SerialManager._on_data_received() 自动识别
if data[0] in (0x00, 0x01):  # 标准/扩展帧
    self.can_data_received.emit(data)
```

## I2C 面板

**源文件**: `src/APP/protocol/i2c_panel_simple.py`

```
功能:
  设备地址输入 (7位)
  寄存器地址输入
  数据编辑 (多字节十六进制)
  → send_command(0x70, [dev_addr, reg_addr, ...data])

OLED 快捷操作:
  0x73: 初始化
  0x74: 清屏
  0x75: 全亮测试
  0x76: 显示文本 — 输入 X/Y 坐标 + 文本
```

## SPI 面板

**源文件**: `src/APP/protocol/spi_panel_simple.py`

```
配置:
  0x80: CPOL/CPHA 模式 + 速率

Flash 操作:
  0x82: 读 JEDEC ID (快捷按钮)
  0x83: 读 Flash — 地址 + 字节数
  0x84: 写 Flash — 地址 + 数据
  0x85: 扇区擦除 — 4KB
  0x86: 全片擦除

通用 SPI:
  0x81: 自定义字节序列收发
```

## DS18B20 面板

**源文件**: `src/APP/protocol/ds18b20_panel.py`

```
操作:
  0xA0: 单次读取 — 显示温度值
  0xA1: 连续监控 — 设置间隔，自动循环读取
  0xA2: 停止监控

温度显示:
  - 界面实时更新温度值
  - FPGA 端同步显示在数码管上
  - 温度数据通过 CH340 → uart_tx_mux DS18B20 通道回传
```
