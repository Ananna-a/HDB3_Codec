# 项目概述

## 项目身份

**多功能协议调试器（Multi-Function Protocol Debugger）** — 基于高云 FPGA 的嵌入式仪器平台，将示波器、函数发生器、协议分析仪等 5 大仪器功能集成到单一硬件，通过 USB CDC 命令链路 + 以太网 UDP 数据链路实现 PC 上位机控制。

| 属性 | 值 |
|------|-----|
| FPGA 芯片 | GW5AT-138B（GW5AT-LV138PG484AC1/I0） |
| 开发板 | ACX720 |
| 综合工具 | GowinSyn（高云云源软件 EDA） |
| 工程入口 | `acm2108_ddr3_CDC.gprj` |
| 顶层模块 | `debugger_top` |
| 引脚约束 | `src/acm2108_ddr3_CDC.cst`（441 行） |
| 系统时钟 | 50 MHz 外部晶振 |
| 上位机框架 | Python 3 / PySide6 + pyqtgraph |

| 资源 | 型号 / 规格 | 用途 |
|------|-----------|------|

| CAN 收发器 | SIT1042AQT/3 | CAN 2.0A/B 总线 |
| OLED | SSD1306 (I2C 0x3C) | 板载 128×64 显示 |

## 通信架构总览

```
┌──────────────────────────────────────────────────────────────┐
│  上位机 (Python / PySide6)                                    │
│  main_app.py → 5 Tab + 调试日志                               │
│  示波器 │ DDS │ 协议转换 │ 逻辑分析 │ 波特图                   │
└──────┬──────────┬──────────────┬────────────────────────────┘
       │ CDC COM  │ CH340 COM    │ UDP :6102
       ▼ (命令)   ▲ (应答+数据)   ▲ (ADC波形)
┌──────────────────────────────────────────────────────────────┐
│  FPGA GW5AT-138B (debugger_top.v, 4157 行)                    │
│  FX2_CDC_Core → fifo_in → cdc_cmd_parser → 60+ 命令路由      │
│    ├── DDS_Module_Dual (② 函数发生器)                         │
│    ├── ADC → DDR3 → eth_udp (① 示波器)                        │
│    ├── CAN/I2C/SPI/PWM/SEQ (③ 协议转换器)                     │
│    └── logic_analyzer_capture (④ 逻辑分析仪)                   │
└──────────────────────────────────────────────────────────────┘
```

三条物理通道（详见 `02_系统架构.md` 和 `03_通信协议.md`）：

| 通道 | 方向 | 物理接口 | 速率 | 用途 |
|------|------|---------|------|------|
| CDC | PC → FPGA | FX2 USB CDC | USB 2.0 FS | 命令下发 |
| CH340 | FPGA → PC | CH340 UART | 115200 bps | 应答 + 数据上报 |
| UDP | FPGA → PC | RGMII 以太网 | 1000 Mbps | 示波器 ADC 波形 |

## 构建命令

### FPGA 端

安装**高云云源软件（Gowin EDA）**后，打开 `acm2108_ddr3_CDC.gprj` 工程：
1. 综合（Synthesize）
2. 布局布线（Place & Route）
3. 生成比特流（Generate Bitstream）
4. 下载到 FPGA

> 工程注册 78 个 `.v` 源文件，使用 GowinSyn 综合引擎。新增文件需在 `.gprj` 中注册路径。

### 上位机端

```powershell
pip install PySide6 numpy pyqtgraph pyserial
cd src\APP
python main_app.py
```

**必须在 `src/APP/` 目录下运行**（模块相对导入依赖 `from core.serial_manager import ...`）。

可选依赖：
- `scipy` — AWG 编辑器三次样条插值
- `Cython + Visual Studio` — UDP 接收 C 扩展加速（3-5 倍性能提升）

## 技术要点

- **双串口分离**：CDC 下行命令 + CH340 上行应答，避免半双工冲突
- **DDR3 乒乓缓冲**：示波器通过 DDR3 双端口实现写入/读出并发，支持边写边读流式传输
- **UDP 高速传输**：FPGA 端 GMII→RGMII，上位机端 Cython 加速接收
- **DDS 32 位相位累加器**：±0.03 Hz 频率精度，5 种基础 ROM + 任意波形 RAM
- **模块解耦**：FPGA 端 cmd_code 区间分发，上位机端 PySide6 信号/槽，Tab 间无直接依赖
- **UART 应答 7 级优先级仲裁**：Bode > 频率 > CAN_RX > DSA/SPI > DS18B20 > 应答帧 > 蓝牙
