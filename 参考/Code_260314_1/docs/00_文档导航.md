## 阅读路径推荐

### 新手入门（最短路径）

```
00 → 01 → 02 → 03 → 16
```

先理解项目是什么、怎么跑起来，再看架构和协议。

### 深入 FPGA 端

```
00 → 02 → 03 → 04（顶层）→ 09（时钟/IP）
  → 05（DDS）→ 06（示波器）→ 07（协议）→ 08（逻辑分析）
```

### 深入上位机端

```
00 → 02 → 03 → 10（架构）→ 11（示波器）→ 12（DDS）
  → 13（协议）→ 14（逻辑分析）→ 15（波特图）
```

### 开发者（需要改代码）

```
全部 18 篇 → AGENTS.md
```

## 文档约定

| 约定 | 说明 |
|------|------|
| `monospace` | 文件名、信号名、寄存器名、命令码 |
| **粗体** | 关键概念、模块名 |
| `→` | 数据流向 |
| `src/path/file.v:NNN` | 源码文件引用（:后面的数字是近似行号） |
| ⚠️ | 注意事项 |
| 🔥 | 关键实现细节 |
