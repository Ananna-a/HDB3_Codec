## Phase 1: FPGA 通信底层 ✅ 已实现

### Task 1: UART 字节收发模块

- [x] **uart_byte_rx.v** — 16 倍过采样，6 次中间采样多数表决抗噪
  - 参数: `CLK_FREQ=50_000_000`, `BAUD_RATE=115200`, `OVERSAMPLE=16`
  - 分频: `BPS_LIMIT = 50M/(115200×16) ≈ 27`
  - 路径: `src/comm/uart_byte_rx.v`

- [x] **uart_byte_tx.v** — 标准 8N1，LSB 先发
  - 分频: `BPS_LIMIT = 50M/115200 ≈ 434`
  - 路径: `src/comm/uart_byte_tx.v`

### Task 2: packet_parser.v ✅ 已实现

- [x] 8 状态 FSM: IDLE→HEAD2→CMD→LEN_L→LEN_H→PAYLOAD→CS→DONE/ERROR
- [x] 帧头重叠保护（S_HEAD2 中收到 0x55 保持在当前状态）
- [x] 校验和计算: `cs_calc = cmd + len_l + len_h + Σpayload`
- [x] payload 逐字节输出（`payload_valid` + `payload_data`），顶层自行存入内部数组
- 路径: `src/hdb3/packet_parser.v`

### Task 3: response_tx.v ✅ 已实现

- [x] 通过 `payload_addr` 从顶层 `sym_buf` 逐字节取数并组帧发送
- [x] 应答帧: `AA 55 | cmd | status | len_l | len_h(0x00) | payload | cs`
- [x] 4 状态: S_IDLE → S_SEND → S_WAIT → S_DONE
- 路径: `src/hdb3/response_tx.v`

## Phase 2: FPGA 核心功能模块 ✅ 已实现

### Task 4: hdb3_encoder.v ✅ 已实现

- [x] 流式分段批处理（S_READ → S_FLUSH/S_SUBST），短延迟缓存处理四零替换
- [x] AMI 交替极性（`ami_pol`）+ B00V/000V 四零替换（`pulse_parity` 奇偶判断）
- [x] 接口: `bit_buf_byte`(in) / `bit_buf_addr`(out) 请求式读取
- 路径: `src/hdb3/hdb3_encoder.v`

### Task 5: hdb3_decoder.v ✅ 已实现

- [x] 批处理模式，3 状态: IDLE→DECODE→DONE
- [x] 解码规则: 仅 +1(3'd1)/-1(3'd2) → bit=1，其他全部 → bit=0
- [x] 接口: `sym_in[7:0]`(in) + `sym_avail`(in) 来自顶层 sym_buf
- 路径: `src/hdb3/hdb3_decoder.v`

### Task 6: dac_playback.v + dac_wave_ram.v ✅ 已实现

- [x] **dac_wave_ram.v**: `reg [7:0] mem[0:2047]` 同步写/读，综合推断 BSRAM
- [x] **dac_playback.v**: 例化 2× dac_wave_ram，采样时钟 50M/250 = 200kHz，符号速率 100kHz，DA1_Clk 与 DA0_Clk 同步
- [x] 电压映射: 0→0x80(0V), +1/+V/+B→0x00(正电压), -1/-V/-B→0xFF(负电压)
- [x] 播放控制: load_done 后预取两拍 → 地址循环 → stop 停止归零
- 路径: `src/hdb3/dac_playback.v`, `src/hdb3/dac_wave_ram.v`

## Phase 3: FPGA 顶层集成 ✅ 已实现

### Task 7: hdb3_top.v ✅ 已实现

- [x] 10 状态主控 FSM: IDLE→ENC_PREP→ENC_RUN→ENC_LOOP→DEC_PREP→DEC_RUN→START_DAC→RESP_WRITE→RESP_SEND/ERR_SEND
- [x] 3 个内部存储数组: `payload_buf[0:63]`, `sym_buf[0:63]`, `bit_buf[0:7]`
- [x] 编码回环: encoder 输出 → DA0 + sym_buf → decoder → DA1
- [x] 解码直通: payload 符号 → DA0 + decoder → DA1 + sym_buf
- [x] 应答: response_tx 按 `payload_addr` 从 `sym_buf` 取数 → AA 55 帧 → UART TX
- [x] LED: led0=2Hz 心跳灯，led1~led7 用于调试 RX/命令/FSM/TX/DAC/应答
- 路径: `src/hdb3_top.v`
